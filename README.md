# ScreenSanctum

**Share your screen, not your secrets.**

ScreenSanctum is an offline-first, cross-platform screenshot redaction tool designed to protect your privacy before you share images.

## What is ScreenSanctum?

ScreenSanctum helps you automatically detect and redact sensitive information from screenshots and images before sharing them. Whether it's emails, phone numbers, credit cards, or other personal data, ScreenSanctum keeps your secrets safe.

## Core Principles

- **Offline-Only**: All processing happens locally on your machine. No cloud, no servers, no internet required.
- **Zero Telemetry**: We don't collect, track, or transmit any data. Ever.
- **One-Screen UX**: Simple, intuitive interface that doesn't get in your way.
- **Cross-Platform**: Works on Windows, macOS, and Linux.

## Features (Planned)

- Automatic detection of sensitive information:
  - Email addresses
  - Phone numbers
  - Credit card numbers
  - Custom patterns
- Interactive redaction editing
- Multiple export formats
- Configurable detection rules

## Installation

```bash
# Clone the repository
git clone https://github.com/woozyrabbit123/screen-sanctum.git
cd screen-sanctum

# Install dependencies
pip install -e .

# Run the application
screensanctum
```
